# 称重列表重复数据问题调研总览

**调研日期**：2026-07-13  
**现象环境**：MaterialClient.Recycle / AttendedWeighing 称重主界面左侧记录列表  
**相关日志**：`repos/MaterialClient/src/MaterialClient.Recycle/bin/.../Logs/2026/07/10/MaterialClient.Recycle-20260710.log`  
**相关 UI**：`WeighingRecordListView.axaml`（绑定 `ListItems`）

---

## 结论摘要

列表出现「相同记录两条、刷新后消失」**不是**称重稳定时重复建库，也**不是** XAML 绑定错误，而是：

**ReactiveUI `MessageBus` 迁移为 ABP `ILocalEventBus` 后，会改 `ListItems` 的订阅回调未切回 UI 线程。**

| 项目 | 说明 |
|------|------|
| 触发时机 | 称重记录创建等事件经 `ILocalEventBus` 通知 ViewModel 刷新列表 |
| 直接原因 | 发布在称重异步队列（线程池），订阅里直接 `Clear`/`Add` `ObservableCollection` |
| 表现 | Avalonia `ItemsControl` 出现幽灵重复行；手动刷新走 UI 线程后恢复正常 |
| 修复状态 | 已修复（见 [01-问题分析与修复.md](./01-问题分析与修复.md)） |
