## 第 1 章 引言

### 1.1 目的

本文档说明了工程渣土资源化利用管理系统提供的对接方法和接口协议，用于实现数据对接、信息共享。第三方开发人员可以使用本文档实现与平台数据共享和对接。

### 1.2 读者对象

本文档的预期读者为需求经理、开发经理、开发工程师、测试工程师等相关人员。

## 第 2 章 开放接口

> **公共说明**
>
> - 除特别注明外，批量新增接口请求体最外层结构均为 **JSON Array**。
> - Header 鉴权内容见 [附录](#附录header-鉴权参数说明)。
> - 图片类字段传 **Base64** 时勿带 Data URL 前缀；例如 `data:image/jpeg;base64,/9j/4AAQSkZJ` 只传 `/9j/4AAQSkZJ`。
> - 多图/多附件字段使用英文逗号 `,` 分隔。

## 附录：Header 鉴权参数说明

### 1. Header 中鉴权参数说明

| 参数名 | 是否必填 | 类型 | 说明 |
|--------|----------|------|------|
| `X-AKZTJG-HMAC-SIGNATURE` | 是 | string | API 输入参数签名结果 |
| `X-AKZTJG-HMAC-ALGORITHM` | 是 | string | 签名的摘要算法，当前仅支持 `hmac-sha256` |
| `X-AKZTJG-HMAC-ACCESS-KEY` | 是 | string | 分给应用的 accessKey，例如：`12345678` |
| `X-AKZTJG-DATE-TIME` | 是 | string | 时间戳，时区为 GMT+8，格式为：`Tue, 09 Nov 2021 08:49:20 GMT`。服务端允许客户端请求最大时间误差为 100 秒 |

### 2. 签名生成代码示例（Java）

```java
// 时间戳格式（RFC 1123）
private static final String DATE_FORMAT = "EEE, dd MMM yyyy HH:mm:ss z";
private static final TimeZone GMT_TIME_ZONE = TimeZone.getTimeZone("GMT");
private static final Locale US_LOCALE = Locale.US;

/**
 * 生成符合平台要求的 HMAC 认证请求头
 *
 * @param urlStr   完整的 API 请求地址（需包含协议头和查询参数）
 * @param method   HTTP 请求方法（需大写：GET、POST 等）
 * @param accessKey 平台颁发的访问密钥 ID
 * @param secretKey 平台颁发的加密密钥
 */
public static Map<String, String> generateAuthHeaders(
        String urlStr, String method, String accessKey, String secretKey) throws Exception {

    // 解析 URL
    URL url = new URL(urlStr);
    URI uri = new URI(url.getProtocol(), url.getHost(), url.getPath(), url.getQuery(), null);

    // 处理查询参数
    List<String> queryParams = new ArrayList<>();
    if (StringUtils.isNotEmpty(uri.getQuery())) {
        for (String param : uri.getQuery().split("&")) {
            String[] kv = param.split("=", 2);
            String encodedKey = URLEncoder.encode(kv[0], "UTF-8");
            String encodedValue = kv.length > 1 ? URLEncoder.encode(kv[1], "UTF-8") : "";
            queryParams.add(encodedKey + "=" + encodedValue);
        }
    }
    Collections.sort(queryParams);
    String queryString = String.join("&", queryParams);

    // 生成时间戳
    String date = getCurrentGmtDate();

    // 构造签名字符串
    String signString = String.format("%s\n%s\n%s\n%s\n",
            method.toUpperCase(), queryString, accessKey, date);

    // 计算 HMAC 签名
    Mac hmac = Mac.getInstance("HmacSHA256");
    hmac.init(new SecretKeySpec(secretKey.getBytes(StandardCharsets.UTF_8), "HmacSHA256"));
    byte[] signatureBytes = hmac.doFinal(signString.getBytes(StandardCharsets.UTF_8));
    String signature = Base64.getEncoder().encodeToString(signatureBytes);

    // 构造认证头
    Map<String, String> headers = new LinkedHashMap<>();
    headers.put("X-AKZTJG-HMAC-ACCESS-KEY", accessKey);
    headers.put("X-AKZTJG-HMAC-SIGNATURE", signature);
    headers.put("X-AKZTJG-HMAC-ALGORITHM", "hmac-sha256");
    headers.put("X-AKZTJG-DATE-TIME", date);
    return headers;
}

/**
 * 获取当前 GMT 时间戳
 */
public static String getCurrentGmtDate() {
    SimpleDateFormat sdf = new SimpleDateFormat(DATE_FORMAT, US_LOCALE);
    sdf.setTimeZone(GMT_TIME_ZONE);
    return sdf.format(new Date());
}

public static void main(String[] args) throws Exception {
    String urlStr = "http://127.0.0.1:8036/dataCenter/v1/extraConnectBill/add";
    String method = "POST";
    String accessKey = "1111";
    String secretKey = "2222";

    Map<String, String> stringStringMap = generateAuthHeaders(urlStr, method, accessKey, secretKey);
}
```

**签名字符串格式**：

```
{HTTP_METHOD}\n{sorted_query_string}\n{accessKey}\n{GMT_date}\n
```
