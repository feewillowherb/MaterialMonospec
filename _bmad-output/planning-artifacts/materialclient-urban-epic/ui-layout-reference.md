# UI 布局参考：MaterialClient.Urban 单界面

**布局草稿来源**（MaterialClient 子仓库）：

`repos/MaterialClient/MaterialClient.Demo/Views/WeighingSystemWindow.axaml`

**窗口标题（Demo）**：凡东城管地磅系统 · 1280×800 · 深色标题栏 + 浅色内容区

## 应用壳约束

| 约束 | 说明 |
|------|------|
| 界面数量 | **仅一个主窗口**，启动即进入称重系统界面 |
| 无登录页 | 不实现登录 Window / 导航至登录 |
| 无授权页 | 不实现授权配置 UI；静态授权文件在**启动时后台读取**，仅写日志 |
| Urban 专用 | 固定 `WeighingMode = 201`、`ProductCode = 5030` |

## 布局结构（与 Demo 对齐）

```
┌──────────────────────────────────────────────────────────────────────────┐
│ Row0 标题栏 #1E293B │ Logo + 标题 │ （Urban：精简菜单，无「退出登录」） │ ─ ✕ │
├──────────────────────────────────────────────────────────────────────────┤
│ Row1 称重状态区 │ 装饰区 │  0.00 吨（大号）  │  称重已结束 / 状态文案   │
├───────────────────────────────────────────────┬──────────────────────────┤
│ 左侧主区（*）                                  │ 右侧栏 288px「照片」      │
│ · Tab：全部记录 / 正常 / 异常                  │ · 车牌识别抓拍            │
│ · 筛选：称重时间、车牌号码 + 搜索/重置         │ · 摄像头抓拍              │
│ · 表头：车牌 | 称重时间 | 重量 | 状态 | 操作   │                          │
│ · 记录列表（WeighingRecord 绑定）              │                          │
│ · 分页                                         │                          │
├──────────────────────────────────────────────────────────────────────────┤
│ Row3 状态栏 │ 设备状态圆点 + 设备名 + 状态文案（DeviceStatus 列表）        │
└──────────────────────────────────────────────────────────────────────────┘
```

## Demo → Urban 差异（实施时注意）

| Demo 元素 | Urban 处理 |
|-----------|------------|
| 顶栏「系统设置 / 项目信息 / 数据同步 / 退出登录」 | 首期可隐藏或仅保留与 Urban 相关的项；**移除登录/退出登录** |
| 「审批」操作按钮 | 按 Urban 业务决定是否保留（无 waybill 时可能改为「详情」或移除） |
| 数据绑定 | 使用正式 `WeighingRecord` ViewModel，非 Demo `Models` |
| 技术栈 | Avalonia 11 + ReactiveUI，与 MaterialClient 主程序一致 |
