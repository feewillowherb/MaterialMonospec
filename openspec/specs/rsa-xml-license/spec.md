# RSA XML License Specification

## Purpose

定义基于 RSA XML 文件的离线授权解密能力，包括 XML 文件解析、RSA 解密、授权过期验证及结果数据结构。

> **Note:** All requirements in this capability have been removed and superseded by JWT-based offline authorization (`jwt-offline-license` and `jwt-license-generation`). This spec is retained for historical reference.

## Requirements

*(No active requirements — all requirements removed in favor of JWT-based offline authorization.)*
