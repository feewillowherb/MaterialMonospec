# OpenSpec 规范说明

本目录包含 MaterialClient 项目的当前能力规范。

## 目的

本目录中的规范定义 MaterialClient 系统的功能与能力。每份规范遵循 OpenSpec 格式，并经过正式评审与批准流程。

## 目录结构

```
specs/
├── <能力名称>/
│   ├── spec.md         # 能力规范
│   ├── design.md       # 设计文档（可选）
│   └── tasks.md       # 实现任务
└── _template/         # 规范模板
```

## 创建新规范

1. 从 `_template/` 复制模板
2. 以能力名称创建新目录
3. 填写规范文档
4. 遵循 OpenSpec 评审流程

## 模板

使用 `_template/` 中的模板以保证一致性。

## 状态

本目录下所有规范均为**现行**并持续维护。

已归档规范请参见 `openspec/archive/legacy/specs/`。
